# PHP-DS Demo: Task Manager System

Requires: php-ds extension (ext/ds).

## Files
- `Task.php` — value object for tasks
- `TaskManager.php` — uses Map, Set, Queue, PriorityQueue, Deque
- `demo.php` — runs a sample scenario (enqueue, prioritize, process, undo)
- `structures_smoke.php` — tiny examples for each DS

## Run
```bash
php demo.php
php structures_smoke.php
```
